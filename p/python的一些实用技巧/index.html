# 1. 脚本中出现中文时报错

python2默认是不识别中文编码的，头部需要这么写

1
2

#!/usr/bin/env python
# -*- coding: utf-8 -*-

python3默认就支持UTF-8了，这个也就不需要了。

# 3. 判断一个包是否已经安装 

1

4

6
7

#!/usr/bin/env python
# -*- coding: utf-8 -*-
import os

output = os.popen('rpm -q jdk1.8').readlines()
for line in output:
    print(line)

os.popen('cmd').readlines()和os.system('cmd')的区别在于前者会把命令的输出保存到output中，而后者则不会，所以如果需要关注命令的输出，就用前面这个。
